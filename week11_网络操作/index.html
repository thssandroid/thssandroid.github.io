    作业指南

    掌握使用安卓调用RESTful API的方法

    案例分析（socket）

    案例程序下载

    功能介绍

    实现对应的client端与server端进行通信。

    点击按钮后，server端将从读取editText中的内容并发送到client端。

    client端在接收到内容后将文本追加显示到textView中。

    代码讲解

    创建新的socket类作为client

    连接访问本地地址127.0.0.1和端口6666

    创建输入流

    Socket client = new Socket();
SocketAddress address = new InetSocketAddress("127.0.0.1", 6666);
client.connect(address);
InputStream is = client.getInputStream();
byte[] bytes = new byte[1024];
    一直对输入的字符串进行获取并打印输出，调runOnUiThread进行输出内容的持续动态添加。 
    while(clientFlag){
    int n = is.read(bytes);
    if(n>0){
        final String text = new String(bytes,0,n);
        System.out.println(text);
        runOnUiThread(new Runnable() {
            @Override
            public void run() {textView.append(text);}
        });
    }
}client.close();

    创建Server端的socket并进行字符串的输出。 

    ServerSocket serverSocket = new ServerSocket(6666);
Socket server = serverSocket.accept();
OutputStream os = server.getOutputStream();
while (serverFlag) {
    if (flag) {
        flag = false;
        String text = editText.getText().toString();
        os.write(text.getBytes());
    }
}
server.close();
serverSocket.close();

    案例分析（network）

    案例程序下载

    功能介绍

    点击按钮后，从输入中读取用户名和密码，并向指定url发送请求，将请求结果显示到TextView中。

    分别使用HttpsURL Connection（右图1）和okhttp（右图2）进行实现。

    代码讲解

    新建HttpsURLConnection连接并进行输入输出和缓存设置 

    HttpsURLConnection conn = (HttpsURLConnection) url.openConnection();
conn.setDoOutput(true);
conn.setDoInput(true);
conn.setUseCaches(false);
    设置请求方式（GET/POST）和内容格式 
    conn.setRequestMethod("POST");
conn.setRequestProperty("Charset", "UTF-8");
conn.setRequestProperty("content-type", "application/json");
    向输出流写数据 
    OutputStream outputStream = conn.getOutputStream();
outputStream.write(outputStr.getBytes());
outputStream.close();
    从输入流读取网络返回的内容 
    InputStream inputStream;
if (conn.getResponseCode() == 200) {
    inputStream = conn.getInputStream(); 
} else {
    inputStream = conn.getErrorStream(); 
}
    使用InputStreamReader 类的buffer获取读取的字符串。 
    InputStreamReader inputStreamReader = new InputStreamReader(inputStream);
BufferedReader bufferedReader = new BufferedReader(inputStreamReader);
StringBuilder buffer = new StringBuilder();
String str;
while ((str = bufferedReader.readLine()) != null) {buffer.append(str);}
    释放所有相关资源。 
    bufferedReader.close();
inputStreamReader.close();
inputStream.close();
conn.disconnect();
    创建OkHttp 的client并将数据类型按json格式解析。 使用Request类进行请求的创建。 
    OkHttpClient client = new OkHttpClient();
MediaType JSON = MediaType.parse("application/json; charset=utf-8");
@SuppressWarnings("deprecation") RequestBody body = RequestBody.create(JSON, jsonStr);
Request request = new Request.Builder()
        .url(requestUrl)
        .post(body)
        .build();

    Client进行队列请求并获取返回值， 根据不同返回值进行成功响应或失败的相关操作。 

    client.newCall(request).enqueue(new Callback() {
    @Override
    public void onFailure(@NotNull Call call, @NotNull IOException e) {}
    @Override
    public void onResponse(@NotNull Call call, @NotNull final Response response) throws IOException {
        Message msg = new Message();
        msg.obj = Objects.requireNonNull(response.body()).string();
        handler.sendMessage(msg);
    }
});

    作业说明

    作业要求

    本次作业的主要任务是实现校园论坛的注册和登陆功能。

    本次作业需要实现以下功能：

    1. 和同组成员一同设计用于登陆的后端API（不限制开发语言及框架，可以使用Django，Spring Boot等框架搭建），该后端API可在大作业中继续使用。

    2. 编写注册页面和登陆页面，实现账号注册和用户登陆功能。

    3. 利用设计好的后端API，使用Android的网络操作，实现注册和登录功能的前后端交互。在后端的数据库中记录账号信息，供登录时使用。可以使用OkHttp等第三方网络库。

    4. 使用SharedPrefs保存登陆状态，在关闭应用再打开时可以恢复上次的登陆状态。
