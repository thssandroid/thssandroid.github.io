    任务1：Java开发环境搭建

    作业指南

    本课程要求使用Java语言进行安卓应用程序的开发。因此首先需要进行Java编程环境的搭建。

    我们为各位同学提供Windows系统和Mac系统配置Java编程环境，编译运行Java程序的教程进行参考。

    Windows系统配置教程

    Mac系统配置教程

    注意：建议同学们下载JDK 8 的最新版本（8u321）进行Java和安卓程序开发，我们也会使用该版本的JDK编译运行作业程序。

    在配置好Java编程环境后，可以使用自己喜欢的IDE（如VS Code，IntelliJ）进行Java开发，这些IDE的环境配置教程均可在网上搜索到。

    此外，请自行学习上述链接中的-classpath方法和jar打包方法。

    作业说明

    在自己的电脑上进行Java环境配置，编写并运行小程序，实现输出自己的姓名和学号的功能，将运行结果截图提交到网络学堂。
