    第十一次作业：Service

    作业目标

    掌握使用安卓Service进行后台服务的方法

    案例分析

    案例程序下载

    功能介绍

    1.点击 开始/停止 按钮：服务start，音乐开始播放

    2.点击 开始/停止 按钮：服务stop，音乐停止播放

    3.点击 开始/停止 按钮：服务重新start，音乐从头开始播放

    4.点击 暂停/播放 按钮：音乐暂停播放

    5.点击 暂停/播放 按钮：音乐继续从上次暂停的地方播放

    代码讲解

    使用bindService()绑定服务

    通过startService() 启动后台服务调用onStartCommand()

    通过stopSelf()或者stopService()停止服务

    对按钮的响应函数及对应的服务的生命周期函数进行完善

    该函数完成启动，停止音乐播放器的功能。启动指音乐从头播放，停止指音乐播放器停止播放且释放相应资源。

    public void startPlayer(View view) {
    Intent intent = new Intent(getBaseContext(), MusicService.class);
    if(!isPlay) {
        intent.putExtra("isPlay",true);
        startService(intent);
    }else{
        stopService(intent);
    }
    isPlay = !isPlay;
}

    该函数完成暂停，继续播放音乐的功能。暂停指音乐停止播放，服务继续运行，继续播放指音乐播放器从上次停止播放的位置继续播放音乐。

    public void pausePlayer(View view) {
    Intent intent = new Intent(getBaseContext(), MusicService.class);
    if(!isPause){
        intent.putExtra("isPlay",true);
        intent.putExtra("isPause",true);
        startService(intent);
    }else{
        intent.putExtra("isPause",false);
        startService(intent);
    }
    isPause = !isPause;
}

    使用bindService()绑定服务，类似于客户端（组件）-- 服务器（服务）模式，提供接口，以便组件与服务进行交互

    private MusicBinder mBinder = new MusicBinder();
public class MusicBinder extends Binder {
    MusicService getService() {
        return MusicService.this;
    }
}
@Override // onBind()方法需返回Ibinder，Ibinder是与服务进行通信的接口。
public IBinder onBind(Intent intent) {return mBinder;}
@Override // 停止时，依次调用onUnbind->onDestroy方法。
public boolean onUnbind(Intent intent) {return super.onUnbind(intent);}
