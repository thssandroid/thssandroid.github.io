    任务4：隐式Intent

    作业目标

    掌握使用隐式Intent打开其他APP并传递数据（打开浏览器、分享文字图片等）的方法

    案例分析：ImplicitIntents

    案例程序下载

    功能介绍

    应用程序包含三个用于发送隐式Intent的按钮： - 在浏览器中打开URL - 在地图上找到位置 - 共享文本字符串，使用IntentBuilder

    点击三个按钮发送隐式Intent，执行不同功能，截图如下：

    代码讲解

    openWebsite()：从EditText获取URL并发送含有该URL的隐式Intent。

    public void openWebsite(View view) {
    String url = mWebsiteEditText.getText().toString();
    // 解析URI并创建intent
    Uri webpage = Uri.parse(url);
    Intent intent = new Intent(Intent.ACTION_VIEW, webpage);
    // 找到对应的activity并进行启动
    if (intent.resolveActivity(getPackageManager()) != null) {
        startActivity(intent);
    } else {
        Log.d("ImplicitIntents", "Can't handle this!");
    }
}

    openLocation()：从EditText获取位置信息，并发送含有该位置的隐式Intent。位置可以是任何可搜索的地理位置。

    public void openLocation(View view) {
    String loc = mLocationEditText.getText().toString();
    // 对地址进行解析并传递给新创建的intent.
    Uri addressUri = Uri.parse("geo:0,0?q=" + loc);
    Intent intent = new Intent(Intent.ACTION_VIEW, addressUri);
    if (intent.resolveActivity(getPackageManager()) != null) {
        startActivity(intent);
    } else {
        Log.d("ImplicitIntents", "Can't handle this intent!");
    }
}

    shareText(View view)：分享文本信息，显示一个包含可供共享的选项的应用选择器由{@link ShareCompat.IntentBuilder}类创建隐式Intent。

    public void shareText(View view) {
    String txt = mShareTextEditText.getText().toString();
    String mimeType = "text/plain";
    ShareCompat.IntentBuilder
            .from(this)
            .setType(mimeType)
            .setChooserTitle(R.string.share_text_with)
            .setText(txt)
            .startChooser();
}

    案例分析：ImplicitIntentsReceiver

    案例程序下载

    功能介绍

    应用程序实现URL的接收展示功能。

    点击ImplicitIntents中的open website，除Chrome外出现本app，点击后跳转到本app进行展示。

    代码讲解

    实现其他APP传递过来的URI的接受和显示。

        Intent intent = getIntent();
    Uri uri = intent.getData();
    if (uri != null) {
        String uri_string = getString(R.string.uri_label)+ uri.toString();
        TextView textView = findViewById(R.id.text_uri_message);
        textView.setText(uri_string);
    }

    修改AndroidManifest.xml文件中的intent-filter，添加对应的action，category和data。

    <intent-filter>
    <action android:name="android.intent.action.VIEW" />
    <category android:name="android.intent.category.DEFAULT" />
    <category android:name="android.intent.category.BROWSABLE" />
    <data android:scheme="http" android:host="developer.android.com" />
</intent-filter>

    作业说明

    作业要求

    本次作业的主要任务是通过隐式Intent在校园论坛的动态详情页面中，使用浏览器打开网络链接，使用地图软件显示位置信息，将文本信息分享到其他应用程序。

    在任务3所创建的应用程序的基础上。在两条动态中分别添加一个网址的url和一个位置信息，并添加分享按钮，实现以下功能：

    1. 使用隐式Intent实现点击动态中的url时，使用浏览器打开动态中的url。

    2. 使用隐式Intent实现点击动态中的位置信息时，使用地图软件显示该位置。

    3. 使用隐式Intent实现点击动态旁的分享按钮时，调用其他应用程序显示该文本信息内容（短信，微信等都可）。
