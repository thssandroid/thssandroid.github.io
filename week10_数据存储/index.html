    第九次作业：数据存储

    作业目标

    掌握使用SharePrefs进行应用内数据存储的方法

    案例分析

    案例程序下载

    功能介绍

    点击 COUNT 按钮使数字+1

    点击 BLACK、RED、BLUE、GREEN 按钮改变背景颜色

    点击 RESET 按钮将数字和颜色重置为默认值

    旋转设备，背景颜色和计数都被保留。

    退出并重启 APP，背景颜色和计数都被保留。

    代码讲解

    将设置状态保存到 SharedPreferences 文件中 并在应用程序重新启动时将该数据读入并恢复，使得重启 APP 后设置不会恢复为默认值。

    创建Shared preferences object 以及shared preferences file 。

    private SharedPreferences mPreferences;
private String sharedPrefFile ="com.example.android.hellosharedprefs";

    并在onCreate函数中进行初始化设置

    mPreferences = getSharedPreferences(sharedPrefFile, MODE_PRIVATE);

    reset函数中进行preferences的清除

    SharedPreferences.Editor preferencesEditor = mPreferences.edit();
preferencesEditor.clear();
preferencesEditor.apply();

    onPause函数中进行preferencesEditor的保存

    SharedPreferences.Editor preferencesEditor = mPreferences.edit();
preferencesEditor.putInt(COUNT_KEY, mCount);
preferencesEditor.putInt(COLOR_KEY, mColor);
preferencesEditor.apply();

    作业说明

    作业要求

    1. 为项目添加设置界面，添加一个SettingActivity用于设置样式，在用户设置页面添加进入该页面的按钮。
    2. 在SettingActivity添加字体样式（至少两种不同的西文字体）、字体大小（至少不同的两种字体大小）的设置选项。
    3. 要求在改变字体样式、字体大小的设置后，在信息流页面中展示的信息可以按照如上设置进行动态改变，并保持美观性，UI不错位。
    4. 使用SharedPrefs保存设置，在关闭应用再打开时可以恢复上次时的设置信息。

    提交说明

    请提交以下内容：

    1. 完整的工程文件、源代码及资源文件
    2. apk安装文件

    实验环境

    1. 建议使用最新版Android Studio (版本号：Bumblebee | 2021.1.1)，低版本可能无法打开。
    2. 助教使用AVD版本 Pixel 4 API 30，其余能正常运行版本皆可。
    3. 助教使用gradle版本为5.6.4。
