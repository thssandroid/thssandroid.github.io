    任务6：TabLayout

    作业目标

    掌握使用Fragment和Tab切换子页面的方法

    案例分析

    案例程序下载

    功能介绍

    该应用程序工具栏中创建了三个视图片段，并提供三个tab来进行点击切换。

    代码讲解

    从视图创建tab布局的实例。 

    TabLayout tabLayout = findViewById(R.id.tab_layout);
    设置每个tab的文本。 
    tabLayout.addTab(tabLayout.newTab().setText(R.string.tab_label1));
tabLayout.addTab(tabLayout.newTab().setText(R.string.tab_label2));
tabLayout.addTab(tabLayout.newTab().setText(R.string.tab_label3));
    设置tab以填充整个布局。 
    tabLayout.setTabGravity(TabLayout.GRAVITY_FILL);
    使用PagerAdapter来管理Fragment中的页面视图。 
    final ViewPager viewPager = findViewById(R.id.pager);
final PagerAdapter adapter = new PagerAdapter
    (getSupportFragmentManager(), tabLayout.getTabCount());
    设置listener用于点击事件。 
    viewPager.setAdapter(adapter);
viewPager.addOnPageChangeListener(new TabLayout.TabLayoutOnPageChangeListener(tabLayout));
tabLayout.addOnTabSelectedListener(new TabLayout.OnTabSelectedListener() {
    @Override
    public void onTabSelected(TabLayout.Tab tab) {
        viewPager.setCurrentItem(tab.getPosition());
}
    重写PagerAdapter并进行tab的切换。 
    @Override
public Fragment getItem(int position) {
    switch (position) {
        case 0: return new TabFragment1();
        case 1: return new TabFragment2();
        case 2: return new TabFragment3();
        default: return null;
    }
}
    为当前的fragment进行布局的inflate。 
    public class TabFragment1 extends Fragment {
    public TabFragment1() {//必需的空公共构造函数}
    @Override
    public View onCreateView(LayoutInflater inflater, ViewGroup container,
                             Bundle savedInstanceState) {
        return inflater.inflate(R.layout.tab_fragment1, container, false);
    }
}

    作业说明

    作业要求

    校园论坛中有四个主要页面，分别为动态浏览页面、动态发布页面、动态搜索页面和个人信息设置页面。其中动态页面中分为全部动态和已关注的人的动态两类。

    本次作业的主要任务是通过导航栏、TabLayout和Fragment实现校园论坛应用中不同页面的切换。

    本次作业需要实现以下功能：

    1. 使用底部导航栏及Fragment实现动态浏览页面、动态发布页面、动态搜索页面和个人信息设置页面的导航跳转（四个页面内的内容可以略写，能够区分四个页面的不同即可）。

    2. 在动态列表页面中使用Tab对全部动态和已关注的人的动态两类动态进行分组展示（页面内容可以略写）。
